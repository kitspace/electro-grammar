Electro Grammar

This is a demo of electro-grammar, a lightning fast parser of electronic component descriptions. Electro-grammar defines a grammar of electronic parts which is executed by Nearly. Currently only surface mount resistors, capacitors and LEDs are supported but the grammar could be expanded to include other parts. A function to match the result to a part in the Common Parts Library is also provided.
